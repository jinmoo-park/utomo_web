프로젝트 루트에 `.env` 파일을 생성하고 다음 내용을 추가하세요:

```

```

## 2. 서버 실행

터미널에서 다음 명령어로 서버를 실행하세요:

```bash
npm run server
```

서버는 `http://localhost:3001`에서 실행됩니다.

## 3. 개발 서버 실행

다른 터미널에서 프론트엔드 개발 서버를 실행하세요:

```bash
npm run dev
```

## 4. 사용 방법

1. 두 개의 터미널을 열어서 각각 서버와 프론트엔드를 실행합니다.
2. 브라우저에서 `http://localhost:5173`으로 접속합니다.
3. 진단 신청 폼을 작성하고 제출합니다.
4. 이메일이 `jinmoo@utomo.co.kr`로 전송됩니다.
